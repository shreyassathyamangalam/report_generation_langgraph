# report_generation_langgraph

## Overview

`report_generation_langgraph` is a Python-based project designed for generating reports using language models and graph-based methodologies. This project leverages various libraries to streamline the report generation process, enabling users to create comprehensive reports efficiently.

## Features

- **Multi-Agent Report Generation**: Utilize multiple agents to improve report accuracy and detail.
- **Integration with Langchain and OpenAI**: Harness the power of advanced language models for generating high-quality text.
- **Flexible and Scalable**: Easily extendable to accommodate more features or enhance capabilities.

## Requirements

This project requires Python version 3.13 or higher. To set up your environment, ensure that you have the following dependencies installed:

- `ipykernel >= 6.30.1`
- `langchain >= 0.3.27`
- `langchain-openai >= 0.3.29`
- `langgraph >= 0.6.4`
- `python-dotenv >= 1.1.1`

## Installation

To get started with the `report_generation_langgraph` project, follow these steps:

1. **Clone the repository**:
    ```bash
    git clone https://github.com/yourusername/report_generation_langgraph.git
    cd report_generation_langgraph
    ```

2. **Set up a virtual environment** (Optional but recommended):
    ```bash
    python -m venv venv
    source venv/bin/activate  # On Windows use `venv\Scripts\activate`
    ```

3. **Install dependencies**:
    ```bash
    pip install -r requirements.txt
    ```

4. **Run the notebook**:
    Open `multi_agent_report_gen.ipynb` in Jupyter Notebook or any compatible environment to start generating reports.

## Usage

Once you have set up the project and installed the necessary dependencies, you can begin using the tools provided in the `multi_agent_report_gen.ipynb` notebook. This notebook contains examples and documentation to help you initiate report generation tasks with ease.
